Homework

Create a database and node REST API for a website where users can upload and listen to podcasts.

Design

Overall you have a lot of freedom with this task. We only specify some minimum requirements but you can make it as simple or complex as you like.

Minimum Requirements

  1. There needs to be a MySQL database:
    • There has to be a users table with at least the fields username and email.
    • There has to be a podcasts table with at least the fields title and content. The content is a dummy field since we don't want you to really upload audio.
    • Every podcast entry has a user who created it.

  2. Create a node REST API. Please remember this simple tutorial for designing good endpoints. You need to create at least the following routes:

    HTTP Method Relative URL Description
    GET /api/users Return an array of all usernames.
    POST /api/users Create a new user.
    GET /api/users/:id Retrieve full user information.
    PUT /api/users/:id Modify user information.
    DELETE /api/users/:id Delete a user.
    GET /api/users/:id/podcasts List all podcasts of a user.
    POST /api/users/:id/podcasts Create a new podcast for the given user.
    PUT /api/users/:id/podcasts/:podId Allow modifying the title of a podcast.
    DELETE /api/users/:id/podcasts/:podId Delete a user's podcast.
